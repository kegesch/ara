## Problem
The MCP server doesn't expose `arad_link`, `arad_unlink`, or `arad_graph` as tools. Agents can add entities but can't create relationships between them or visualize the graph through MCP.

## What should happen
Add three more MCP tools:
- `arad_link` — create relationships between entities (with edge type validation)
- `arad_unlink` — remove relationships
- `arad_graph` — get graph visualization (return Mermaid/DOT string)

## Where
`src/mcp/server.ts`
