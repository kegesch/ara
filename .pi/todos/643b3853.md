## Goal
Compile ARAD to a standalone binary for easy distribution — no Node/Bun runtime required.

## Scope
- Bun's `bun build --compile` already produces a single binary
- Test on Windows, macOS, Linux
- Add CI (GitHub Actions) to build release binaries for all three platforms
- Add `bun run build` script (already in package.json)
- Add a `Makefile` or `justfile` for cross-platform builds
- Consider binary size optimization (Bun produces ~80MB binaries — acceptable for a CLI tool)
- Add `--version` output that includes build info

## Why
Zero-dependency installation: download one file, put it in PATH, done. Especially important for CI/CD pipelines where you want `arad check` as a gate.
