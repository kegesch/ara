## Context
The `remove --clean` command (D-021) and `rename` command (D-022) both do graph mutation but have no test coverage. These are the most error-prone commands since they touch multiple entity files.

## What to add
- Tests for `remove` with `--force` and `--clean`
- Tests for `rename` with reference propagation across driven_by, enables, derived_from, conflicts_with, supersedes, promoted_to
- Test edge case: removing an entity that doesn't exist
- Test edge case: renaming to an ID that already exists
