## Context Grouping

Add a `context` field to all entities and support context-scoped operations.

### What
- Add optional `context` field to entity YAML frontmatter
- Add `context` to `AradGraph` with context‚Üíentity mapping
- `arad init` creates a default context (project name from arad.yaml)
- `arad add --context billing "..."` sets context on creation
- `arad list --context billing` scopes listing
- `arad check --context billing` scopes checking
- `arad status` shows breakdown by context
- Context can be set/changed via `arad edit` or a new `arad context` command

### Why
Flat entity lists don't scale. Contexts scope queries and surface per-area health.
