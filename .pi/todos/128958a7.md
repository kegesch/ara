## Problem
There's no `arad rename` command. If you want to change an entity's ID (e.g. renumber it) or fix a title, you have to manually rename the file and update all references across other entities.

## What should happen
`arad rename D-001 D-099` should:
1. Rename the file
2. Update the ID in the frontmatter
3. Find all entities that reference D-001 in their driven_by, enables, etc.
4. Update those references to D-099
5. Re-slug the filename if title changed

## Where
New file: `src/commands/rename.ts`
