## Context
The MCP server now has `arad_link`, `arad_unlink`, and `arad_graph` (D-027), but is missing tools for the new mutation commands added in the dogfooding session.

## Missing tools
- `arad_remove` — remove entity with `--force`/`--clean`
- `arad_rename` — rename entity ID, propagating refs
- `arad_import` — import ADR files
- `arad_status` — project health summary

## Related
- D-021 (remove), D-022 (rename), D-023 (status), D-024 (import)
- R-021 (MCP must expose tools for programmatic graph manipulation)
